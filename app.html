API Endpoints

Image Generation

GET https://flux-schnell.hello-kaiiddo.workers.dev/img?prompt={prompt}&model={model}&guidance={guidance}&strength={strength}

Generates an image based on the provided parameters.

Parameter Required Default Description
prompt Yes - Text description of the image to generate
model No flux-schnell Model to use (flux-schnell, flux-real, flux-anime)
guidance No 8.5 How closely the image follows the prompt (1-20)
strength No 1.4 Strength of the generation (0.1-2)

Example Request

fetch('https://flux-schnell.hello-kaiiddo.workers.dev/img?prompt=Photo%20of%20an%20extremely%20cute%20alien%20fish%20swimming%20an%20alien%20habitable%20underwater%20planet&model=flux-schnell&guidance=8.5&strength=1.4')
  .then(response => response.blob())
  .then(blob => {
    const imageUrl = URL.createObjectURL(blob);
    // Use the image URL
  });

Response

Returns the generated image directly as binary data with content-type image/png.
